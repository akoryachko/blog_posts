Running the notebooks locally on Mac/Unix system:
- Clone the repo.
- Open the repo in VSCode.
- Use terminal to navigate to the `pyspark_to_production` folder.
- Run `make init` command to set up the virtual environment with all the dependencies.
- Run `. .venv/bin/active` to activate the virtual environment and make the kernel available.
- Open a notebook of interest in VSCode.
- Select the notebook kernel `pyspark_to_production_blogpost`.
- Restart VSCode if the Kernel is not visible.
